# 1. dtd 约束

<!ELEMENT shop-cart (item+)>  shop-char 下只能出现 item 节点，+ 表示 item 节点可以出现1次到多次，
<!ELEMENT item (name)> 表示 item 下只能出现 name 标签
<!ATTLIST item tag NMTOKEN #REQUIRED>  item 标签的属性表述，tag 必须出现
<!ELEMENT name (#PCDATA)>  (#PADATA) 表示 name 作为数据标签

<!ELEMENT item (name, age)> 表示必须有 name 和 age 标签

<!ATTLIST 元素名称 属性名称 属性类型 默认值>
<!ATTLIST item tag CDATA #REQUIRED>  表示 item 标签上的 tag 属性，类型为字符串
类型描述
CDATA字符类型
(X1 | X2 | X3)枚举列表中的值
ID唯一 id
IDREF值为另外一个元素的 id
IDREFS值为其他 id 的列表
NMTOKEN值为合法的 XML 名称
NMTOKENS值为合法的 XML 名称的列表
ENTITY值是一个实体
ENTITIES值是一个实体列表
NOTATION此值是符号的名称
xml值是一个预定义的 XML 值

# 2. XML Schema

比 dtd 更复杂，提供了更多功能, 是 dtd 的替代。可以描述 xml 的文档的结构，以 xsd 后缀结尾。 符合 w3c 标准。

<?xml version="1.0" encoding="UTF-8"?>
<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema">
    <xs:element name="shop-cart">
        <xs:complexType>
            <xs:sequence>
                <xs:element name="item" maxOccurs="unbounded">
                    <xs:complexType>
                        <xs:sequence>
                            <xs:element name="name" type="xs:string" />
                             <xs:element name="price">
                                <xs:simpleType>
                                    <xs:restriction base="xs:integer">
                                        <xs:minExclusive value="1"></xs:minExclusive>
                                        <xs:maxExclusive value="10000"></xs:maxExclusive>
                                    </xs:restriction>
                                </xs:simpleType>
                            </xs:element>
                        </xs:sequence>
                        <xs:attribute name="tag" use="required"/>
                    </xs:complexType>
                </xs:element>
            </xs:sequence>
        </xs:complexType>
    </xs:element>
</xs:schema>
  • <xs:complexType> 有子节点时必须使用这个标签
  • <xs:sequence> 表示需要按照子节点顺序的前后进行
  • <xs:element name="xxx" type="xs:string">
  • <xs:attribute name="tag" use="required" /> 表示对属性的描述
  • maxOccurs 最大出现次数
  • minOccurs 表示最小出现次数
  • <xs:simpleType> 表示对基础类型进行限制
